# Postgres Setup

As of Gitlab 19.0, Postgres is no longer included with Gitlab. Please see [Configure the GitLab chart with an external database](https://docs.gitlab.com/charts/advanced/external-db/) documentation for more details.

Bitnami Postgres is currently included in this chart for testing purposes, which is subject to change.  You can find the configuration in [values.yaml](../chart/values.yaml#L421).
